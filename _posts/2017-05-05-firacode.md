If you are developing in Elixir and use the github Atom code editor you might want to
take a look at the [Firacode](https://github.com/tonsky/FiraCode) Font. It has support
for programming ligatures. Font changes can be fairly easily managed in Atom by installing
the [fonts package](https://atom.io/packages/fonts). That package provides dozens of
possibilities for monospaced programmer fonts.
